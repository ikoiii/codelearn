Form dan Input HTML

Form HTML digunakan untuk mengumpulkan input dari pengguna.

Element Dasar Form

<form action="/submit" method="POST"> <!-- Input fields here --> </form>

Atribut penting:

  • action - URL tempat data akan dikirim
  • method - HTTP method (GET atau POST)

Jenis-jenis Input:

1. Text Input

<label for="nama">Nama:</label> <input type="text" id="nama" name="nama" placeholder="Masukkan nama">

2. Password Input

<label for="password">Password:</label> <input type="password" id="password" name="password">

3. Email Input

<label for="email">Email:</label> <input type="email" id="email" name="email" required>

4. Number Input

<label for="umur">Umur:</label> <input type="number" id="umur" name="umur" min="1" max="120">

5. Radio Buttons

<label>Jenis Kelamin:</label> <input type="radio" id="pria" name="gender" value="pria"> <label for="pria">Pria</label> <input type="radio" id="wanita" name="gender" value="wanita"> <label for="wanita">Wanita</label>

6. Checkboxes

<input type="checkbox" id="setuju" name="setuju"> <label for="setuju">Saya setuju dengan syarat dan ketentuan</label>

7. Dropdown/Select

<label for="kota">Kota:</label> <select id="kota" name="kota"> <option value="">-- Pilih Kota --</option> <option value="jakarta">Jakarta</option> <option value="surabaya">Surabaya</option> <option value="bandung">Bandung</option> </select>

8. Textarea

<label for="pesan">Pesan:</label> <textarea id="pesan" name="pesan" rows="4" cols="50"></textarea>

9. File Input

<label for="file">Upload File:</label> <input type="file" id="file" name="file" accept=".jpg,.png,.pdf">

10. Buttons

<input type="submit" value="Kirim"> <input type="reset" value="Reset"> <input type="button" value="Klik Saya"> <button type="submit">Kirim Form</button>

Atribut Input Penting:

  • id - Pengenal unik untuk element
  • name - Nama untuk data submission
  • value - Nilai default
  • placeholder - Teks petunjuk
  • required - Field wajib diisi
  • readonly - Field hanya bisa dibaca
  • disabled - Field dinonaktifkan
  • min/max - Nilai minimum/maximum
  • maxlength - Maksimum karakter

Contoh Form Lengkap:

<form action="/submit" method="POST"> <label for="nama">Nama Lengkap:</label> <input type="text" id="nama" name="nama" placeholder="John Doe" required> <label for="email">Email:</label> <input type="email" id="email" name="email" placeholder="john@example.com" required> <label for="umur">Umur:</label> <input type="number" id="umur" name="umur" min="1" max="120"> <label>Jenis Kelamin:</label> <input type="radio" id="pria" name="gender" value="pria"> <label for="pria">Pria</label> <input type="radio" id="wanita" name="gender" value="wanita"> <label for="wanita">Wanita</label> <label for="kota">Kota:</label> <select id="kota" name="kota"> <option value="">-- Pilih Kota --</option> <option value="jakarta">Jakarta</option> <option value="surabaya">Surabaya</option> <option value="bandung">Bandung</option> </select> <label for="pesan">Pesan:</label> <textarea id="pesan" name="pesan" rows="3" placeholder="Tulis pesan Anda..."></textarea> <button type="submit">Kirim Form</button> </form>
Tips: Selalu gunakan tag <label> dengan atribut for yang sesuai dengan id input untuk aksesibilitas yang lebih baik.
