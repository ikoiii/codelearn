Pengenalan HTML

HTML (HyperText Markup Language) adalah bahasa markup standar untuk membuat halaman web.

Apa itu HTML?

  • HTML bukanlah bahasa pemrograman, melainkan bahasa markup
  • HTML menggunakan tag untuk mendeskripsikan struktur konten
  • HTML adalah dasar dari semua website di internet
  • HTML bekerja sama dengan CSS (untuk gaya) dan JavaScript (untuk interaktivitas)

Struktur Dasar HTML

<!DOCTYPE html> <html> <head> <title>Judul Halaman</title> </head> <body> <h1>Heading Utama</h1> <p>Ini adalah paragraf.</p> </body> </html>

Penjelasan Tag Penting:

  • <!DOCTYPE html> - Mendeklarasikan dokumen sebagai HTML5
  • <html> - Elemen root dari halaman HTML
  • <head> - Berisi metadata tentang dokumen
  • <title> - Judul halaman yang muncul di browser tab
  • <body> - Berisi konten yang terlihat di halaman
  • <h1> - Heading level 1 (paling penting)
  • <p> - Paragraf

Tag Umum lainnya:

<!-- Heading --> <h1>Heading 1</h1> <h2>Heading 2</h2> <h3>Heading 3</h3> <!-- Text Formatting --> <b>Tebal</b> <i>Miring</i> <u>Garis bawah</u> <strong>Penting</strong> <em>Penekanan</em> <!-- Links --> <a href="https://www.example.com">Link ke Example</a> <!-- Images --> <img src="gambar.jpg" alt="Deskripsi gambar"> <!-- Lists --> <ul> <li>Item 1</li> <li>Item 2</li> </ul> <!-- Line Break --> <br> <!-- Horizontal Rule --> <hr>
Catatan: HTML tidak case-sensitive, tetapi lebih baik menggunakan lowercase untuk konsistensi dan HTML5.

Latihan Pertama:

Coba buat file HTML sederhana dengan struktur berikut:

<!DOCTYPE html> <html> <head> <title>Halaman Pertama Saya</title> </head> <body> <h1>Selamat Datang!</h1> <p>Ini adalah halaman web pertama saya.</p> <p>Saya sedang belajar HTML!</p> </body> </html>
